## How to start

* Install Node.JS, npm

* Create file ```.env.local``` 
```
SCENE_URL=url to your scene description
```
* Run command ```npm install``` in project folder
* Run command ```npm start``` in project folder
* Open in browser [localhost:8088](http://localhost:8088/)
